# Do it 딥러닝 입문 교재 실습 부분 정리

### 졸업프로젝트 대비 스터디(2020.02.10~2020.03.02)

----------------

교재는 1~9강으로 구성되어있으며, 실습부분은 3강부터 있다.

#### 목차

- [x] 3강 : 선형회귀
- [x] 4강 : 이진분류 + 단일신경망
- [x] 5강 : 4강의 연장
- [x] 6강 : 다층신경망
- [] 7강 : 다중분류
- [ ] 8강 : 합성곱 신경망
- [ ] ~~9강 : 순환 신경망~~

---------------

프로젝트의 최종 목표가 RNN(순환 신경망)이 아니므로,
이 실습 부분 정리에서는 9강을 뺀 나머지 자료를 올릴 것임.
